# Code of Conduct

DelayEngine is an open source project. Keep discussions respectful, practical, and focused on helping people run safer live streams.

## Expected Behavior

- Be respectful and patient.
- Explain technical tradeoffs clearly.
- Give useful feedback without attacking people.
- Respect privacy, especially around stream keys and channel data.

## Not Accepted

- Harassment or personal attacks.
- Publishing someone else's stream key, private logs, or private channel data.
- Spam or intentionally misleading support.

Maintainers may remove content or restrict participation when needed to keep the project healthy.

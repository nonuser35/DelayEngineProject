# Ferramentas de execução

Os binários do FFmpeg, FFprobe e MediaMTX não são mantidos no histórico do repositório por causa dos limites de tamanho do GitHub.

Para usar o aplicativo pronto, baixe o ZIP completo em [Releases](https://github.com/nonuser35/DelayEngineProject/releases/latest).

Para desenvolvimento, execute `scripts/install-ffmpeg.cmd` para preparar o FFmpeg. O pacote portátil publicado em Releases já contém todas as ferramentas necessárias.
